## 使用环境

### 推荐环境

**Linux (Ubuntu/Debian)**:


**WSL (Windows)**:

## 快速开始

### 编译编译器

```bash
gcc -o miniic *.c
```

### 编译示例

```bash
./miniic demo/hello.c > hello.s
nasm -f elf32 hello.s -o hello.o
gcc -m32 hello.o -o hello
./hello
echo $?  # 查看输出: 42
```
